jQuery.hrmny

Hey there,

this is a demo of the current version of jquery.hrmny

Use the input below to specify any color you like by using the following formates:

  • Hex Example
  • RGB Example
  • HSL Example
.[namespace](-[modifier])-[cssproperty]-[colorname]

You may want to manipulate the classes on the squares using a webinspector and this syntax.
class="hrmny-background-base" for example will set the background color to the base color. While class="hrmny-hover-color-left1" will set the font color to left1 when the element is hovered, focused or active.

Available CSS-Properties*:

  • "background" for "background-color"
  • "color" for "color"
  • "border" for "border-color"

Available Color-Names*:

  • base
  • left1
  • left2
  • right1
  • right2

Available Modifiers*:

  • hover for (:hover, :active, :focus)

* All this Values will be customizable and by default be more complete in future versions
